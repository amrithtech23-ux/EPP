# ⚡ Electric Power Projects Chatbot (EPP)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://streamlit.io)

An AI-powered chatbot specialized in Electric Power Projects, designed for project managers, professionals, students, and job seekers in the electric power industry.

## 🎯 Target Audience

- Project Managers in Electric Power Industry
- Professionals in Electric Power Industry
- B.E. Electrical and Electronics Students
- Job Seekers in Electric Power Industry

## 🚀 Features

- **Domain-Specific Expertise**: Specialized in:
  - Balance of Plant (BoP) Services for Thermal Power Projects
  - Balance of Plant (BoP) Services for Gas Power Projects
  - Turnkey Solutions for Solar Power Plants
  - Ground-mounted Solar Installations

- **Interactive UI**: User-friendly Streamlit interface
- **Smart Suggestions**: 10 random suggested prompts
- **Conversation History**: Maintains context across queries
- **API Integration**: Powered by Qwen chatbot API

## 📋 Prerequisites

- Python 3.8+
- Streamlit
- Qwen API Key (from Alibaba Cloud DashScope)
